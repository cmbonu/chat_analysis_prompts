## AI prompts to analyse user activity in a whatsapp chat group


### How to use
1. Export the chat group messages and store somewhere on youir local machine
2. Open you preferred AI Chatbot.
3. Add chat file to the chatbot context
4. Paste the contents of `structure_chat_logs.md` and run.
5. Paste the contents of `generate_standard_report.md` to get a standard text report
6. [Optional] Check out `sample_adhoc_prompts.md` to explore question you can ask
